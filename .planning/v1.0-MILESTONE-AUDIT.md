## Tech Debt

### 1. Dashboard Widget Two-Step Flow (INTG-01) — Low Priority

**Phase:** 24
**Decision:** Locked architectural decision. Widget navigates to `/journal` but modal doesn't auto-open.
**Fix if desired:** Add `?open=today` URL param to widget's `router.push`, read it in `JournalPageContent` with `useSearchParams` to auto-open modal.

### 2. sendBeacon POST to PATCH-Only Endpoint — Low Priority

**Phase:** 21
**Issue:** `navigator.sendBeacon` always sends POST. For existing entries, it targets `/api/journal/[id]` which has no POST handler → 405 on browser close within debounce window.
**Fix:** Add POST handler to `/api/journal/[id]/route.ts` that delegates to update logic, or accept as edge case.

### 3. Prompt Banner Not Restored on Modal Reopen — Cosmetic

**Phase:** 22
**Issue:** SWR cache returns same object reference for same date → `entry` effect doesn't re-fire → `promptKey` stays null after `open` effect reset.
**Fix:** Seed `promptKey` from `entry` in the `open` effect instead of relying on a separate `entry` effect.
