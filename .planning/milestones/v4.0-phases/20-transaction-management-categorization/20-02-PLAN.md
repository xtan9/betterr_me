All routes follow the established money API auth pattern: `createClient() -> getUser() -> 401 if null -> resolveHousehold()`. Use `{ log } from '@/lib/logger'` for error logging.

**1. `app/api/money/categories/route.ts`** — GET and POST:

- **GET**: Instantiate `CategoriesDB(supabase)`. Call `getVisible(householdId)` to return categories excluding hidden ones. Return `{ categories }`.
- **POST**: Parse body with `categoryCreateSchema`. Instantiate `CategoriesDB(supabase)`. Call `create({ ...parsed.data, household_id: householdId, is_system: false })`. Return `{ category }` with status 201.

**2. `app/api/money/categories/[id]/route.ts`** — PATCH and DELETE:

- **PATCH**: Parse body with `categoryUpdateSchema`. Get category by ID first to verify it exists and `is_system === false` (cannot update system categories). Call `update(id, parsed.data)`. Return `{ category }`.
- **DELETE**: Get category by ID. Verify `is_system === false` (400 if system). Call `delete(id)`. Return `{ success: true }`.

**3. `app/api/money/categories/hidden/route.ts`** — POST and DELETE:

- **POST**: Accept `{ category_id: string }`. Call `categoriesDB.hide(householdId, category_id)`. Return `{ success: true }`.
- **DELETE**: Accept `{ category_id: string }` via request body (or query param). Call `categoriesDB.unhide(householdId, category_id)`. Return `{ success: true }`.

**4. `app/api/money/merchant-rules/route.ts`** — GET and POST:

- **GET**: Instantiate `MerchantRulesDB(supabase)`. Call `getByHousehold(householdId)`. Return `{ rules }`.
- **POST**: Parse body with `merchantRuleCreateSchema`. Call `create({ ...parsed.data, household_id: householdId, merchant_name_lower: parsed.data.merchant_name.toLowerCase() })`. Return `{ rule }` with status 201.

**5. `app/api/money/merchant-rules/[id]/route.ts`** — DELETE:

- **DELETE**: Instantiate `MerchantRulesDB(supabase)`. Call `delete(params.id)`. Return `{ success: true }`.

**IMPORTANT:** All routes use try/catch with `log.error('VERB /api/money/... error', error)` and return NextResponse.json({ error: 'Failed to ...' }, { status: 500 }).

**1. `app/api/money/transactions/[id]/route.ts`** — GET and PATCH:

- **GET**: Instantiate `TransactionsDB(supabase)`. Call `getById(params.id)`. Return 404 if null. Also fetch splits via `TransactionSplitsDB.getByTransaction(params.id)`. Return `{ transaction, splits }`.
- **PATCH**: Parse body with `transactionUpdateSchema` from `lib/validations/money.ts`. Call `transactionsDB.update(params.id, parsed.data)`. Return `{ transaction }`.

**2. `app/api/money/transactions/[id]/splits/route.ts`** — GET, POST, DELETE:

- **GET**: Instantiate `TransactionSplitsDB(supabase)`. Call `getByTransaction(params.id)`. Return `{ splits }`.
- **POST**: Parse body with `transactionSplitSchema`. Validate that splits sum to original transaction amount:
  1. Get transaction via `TransactionsDB.getById(params.id)` — 404 if not found
  2. Calculate `splitTotal = splits.reduce((sum, s) => sum + s.amount_cents, 0)`
  3. If `Math.abs(splitTotal) !== Math.abs(transaction.amount_cents)`, return 400 with "Split amounts must equal transaction total"
  4. Delete existing splits via `splitsDB.deleteByTransaction(params.id)`
  5. Create new splits via `splitsDB.create(parsed.data.splits.map(s => ({ ...s, transaction_id: params.id })))`
  6. Return `{ splits }` with status 201
- **DELETE**: Call `splitsDB.deleteByTransaction(params.id)`. Return `{ success: true }`.

**3. Update `app/api/money/transactions/route.ts`** — Enhance GET handler:
