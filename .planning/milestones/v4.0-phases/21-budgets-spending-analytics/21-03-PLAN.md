**BudgetRing** (`components/money/budget-ring.tsx`):

Custom SVG circular progress ring — lightweight, no library needed (~30 lines).

```tsx
"use client";

interface BudgetRingProps {
  percent: number;  // 0-100+ (can exceed 100 for over-budget)
  size?: number;    // default 48
  strokeWidth?: number;  // default 4
  color?: string;   // override auto color
}
```

Implementation:
- Calculate radius, circumference, and strokeDashoffset from percent
- Cap visual arc at 100% (ring fills completely, doesn't overflow)
- Color thresholds following Calm Finance aesthetic:
  - 0-74%: Sage green `hsl(var(--money-sage))` — healthy
  - 75-89%: Amber `hsl(var(--money-amber))` — approaching limit
  - 90%+: Muted coral (add `--money-caution: 15 65% 55%` in light, `15 55% 45%` in dark to globals.css) — near/at limit. NOT aggressive red.
- Background track: `hsl(var(--money-border))`
- Rotate -90deg so arc starts from top
- Add `transition-all duration-500` for smooth animation
- Support `className` prop for layout integration
- When over 100%, ring is fully filled in caution color
