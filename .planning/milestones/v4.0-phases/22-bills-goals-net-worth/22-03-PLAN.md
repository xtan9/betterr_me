  <files>

  <action>
    All routes follow the established money API auth pattern: `createClient -> getUser -> 401 if null -> resolveHousehold -> DB operation`.

    **Goals list and create** (`app/api/money/goals/route.ts`):
    - `GET`: Fetch all active goals via SavingsGoalsDB.getByHousehold(householdId). For each goal, compute projection:
      - For manual-funded goals: get contributions from last 3 months via SavingsGoalsDB.getContributions(goalId), calculate monthly_rate = total / months_with_data, remaining = target_cents - current_cents, months_to_go = remaining / monthly_rate, projected_date = addMonths(new Date(), months_to_go). Guard against zero/negative rate (return null projected_date).
      - For linked goals: compute rate from current_cents change over last 3 months (would need historical snapshots or contribution-like tracking -- simplify: use same contribution-based approach since linked goals auto-update current_cents on sync).
      - Return `{ goals: GoalWithProjection[] }` where GoalWithProjection extends SavingsGoal with `projected_date: string | null`, `monthly_rate_cents: number`, `status_color: 'green' | 'yellow' | 'red'`
      - Status color logic per user decision: green if projected_date <= deadline (or no deadline), yellow if projected_date <= deadline + 30 days, red if projected_date > deadline + 30 days
    - `POST`: Validate body with goalCreateSchema -> convert target_amount to cents (toCents(body.target_amount)) -> SavingsGoalsDB.create({ household_id, name, target_cents, deadline, funding_type, linked_account_id, icon, color }) -> return 201

    **Single goal** (`app/api/money/goals/[id]/route.ts`):
    - `GET`: Get goal by ID with projection (same computation as list)
    - `PATCH`: Validate with goalUpdateSchema -> convert amounts if present -> SavingsGoalsDB.update(id, updates) -> return updated goal
    - `DELETE`: SavingsGoalsDB.delete(id) -> return 204

    **Goal contributions** (`app/api/money/goals/[id]/contributions/route.ts`):
    - `GET`: SavingsGoalsDB.getContributions(goalId) -> return `{ contributions: GoalContribution[] }`
    - `POST`: Validate with contributionCreateSchema -> convert amount to cents -> SavingsGoalsDB.addContribution(goalId, amountCents, note) -> return 201 with contribution

    **Net worth current** (`app/api/money/net-worth/route.ts`):
    - `GET`: Compute current net worth:
      1. Get all accounts via MoneyAccountsDB.getByHousehold(householdId) -- sum positive balances as assets, negative as liabilities
      2. Get manual assets via ManualAssetsDB.getByHousehold(householdId) -- sum value_cents
      3. total_assets = account_assets + manual_asset_total
      4. total_liabilities = abs(sum of negative account balances)
      5. net_worth = total_assets - total_liabilities
      6. Get latest snapshot for change comparison: NetWorthSnapshotsDB.getLatest(householdId)
      7. Compute change: amount_change = net_worth - latest.total_cents, percent_change = (amount_change / abs(latest.total_cents)) * 100
      8. Return `{ net_worth_cents, assets_cents, liabilities_cents, manual_assets_cents, change: { amount_cents, percent, vs_date } }`
      9. Include breakdown by account type: `{ accounts_by_type: { checking: cents, savings: cents, credit: cents, loan: cents, other: cents } }`

    **Net worth snapshots** (`app/api/money/net-worth/snapshots/route.ts`):
    - `GET`: Accept `period` query param (1M, 3M, 6M, 1Y, ALL). Compute fromDate based on period. NetWorthSnapshotsDB.getHistory(householdId, fromDate, toDate) -> return `{ snapshots: NetWorthSnapshot[] }`
      - 1M: 30 days ago, 3M: 90 days, 6M: 180 days, 1Y: 365 days, ALL: no fromDate filter
      - Format dates for chart display: "Jan 2026", "Feb 2026" for months

    **Manual assets** (`app/api/money/manual-assets/route.ts`):
    - `GET`: ManualAssetsDB.getByHousehold(householdId) -> return `{ assets: ManualAsset[] }`
    - `POST`: Validate with manualAssetCreateSchema -> convert value to cents -> create -> return 201

    **Single manual asset** (`app/api/money/manual-assets/[id]/route.ts`):
    - `PATCH`: Validate with manualAssetUpdateSchema -> convert value if present -> update -> return updated
    - `DELETE`: ManualAssetsDB.delete(id) -> return 204
  </action>
