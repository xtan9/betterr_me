Modify `components/journal/journal-editor.tsx`:
1. Import `Placeholder` from `@tiptap/extensions` (same package as CharacterCount, already installed)
2. Add `Placeholder.configure({ placeholder: "Start writing..." })` to the `extensions` array in `useEditor`
3. Accept an optional `placeholder?: string` prop on `JournalEditorProps` and pass it to `Placeholder.configure({ placeholder: placeholder ?? "Start writing..." })`. This allows the parent to customize the placeholder text if needed (though for now it will use the default).

Modify `app/globals.css` — add Placeholder CSS inside the existing `.tiptap-journal` section (after the last `.tiptap-journal` rule for `s`):

```css
.tiptap-journal .tiptap p.is-empty::before {
  @apply text-muted-foreground/50;
  content: attr(data-placeholder);
  float: left;
  height: 0;
  pointer-events: none;
}
```

Use `@apply text-muted-foreground/50` for consistent theming (Tailwind utility, not raw CSS color). The `float: left; height: 0; pointer-events: none` ensures the placeholder does not affect layout or block clicks.
