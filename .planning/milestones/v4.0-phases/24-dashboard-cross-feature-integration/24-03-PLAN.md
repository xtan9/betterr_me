**1. Create `lib/hooks/use-journal-links.ts`:**

SWR hook for fetching enriched links for a journal entry:

```typescript
export function useJournalLinks(entryId: string | null) {
  const { data, error, isLoading, mutate } = useSWR(
    entryId ? `/api/journal/${entryId}/links` : null,
    fetcher
  );
  return {
    links: data?.links ?? [],
    error,
    isLoading,
    mutate,
  };
}
```

Conditional SWR key (null when no entryId) so it doesn't fetch for new entries without an ID yet.

Also add mutation helpers:
- `addLink(entryId: string, linkType: string, linkId: string)` — POST to `/api/journal/${entryId}/links` with body `{ link_type, link_id }`, then mutate
- `removeLink(entryId: string, linkId: string)` — DELETE to `/api/journal/${entryId}/links?link_id=${linkId}`, then mutate

Export these as part of the hook or as separate functions.
