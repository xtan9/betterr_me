- This component is used on the `/money/bills` page as an enhancement to the existing calendar tab

**`components/money/insight-card.tsx`:**
- "use client" component
- Props: `{ insight: Insight; onDismiss: (id: string) => void; }`
- Renders a compact card with Calm Finance styling
- Layout: icon (left) + message (center) + dismiss button (right)
- Icon varies by type:
  - spending_anomaly: TrendingUp (lucide)
  - subscription_increase: CreditCard (lucide)
  - goal_progress: Target (lucide)
  - low_balance_warning: AlertCircle (lucide)
  - bill_upcoming: CalendarCheck (lucide)
- Severity affects subtle accent:
  - "attention": left border using --money-amber
  - "positive": left border using --money-sage
  - "info": no accent border
- Message is rendered from insight.data using i18n translation keys (e.g., t("insights.spendingAnomaly", { category: data.category, percent: data.percent_change, period: data.period }))
- AIML-05: Use anxiety-aware language. NEVER say "overspent" or "danger". Use progress framing:
  - Spending up: "Groceries is up 15% from your 3-month average"
  - Subscription: "Netflix increased from $15 to $23"
  - Goal ahead: "Emergency Fund is 90% of the way there"
  - Goal behind: "Vacation Fund may need a bit more time based on current pace"
- Dismiss button: small X icon, calls onDismiss(insight.id)
