**1. Create `lib/hooks/use-journal-widget.ts`:**

Self-contained SWR hook per project decision ("Dashboard journal widget must be self-contained, own SWR hook, not added to DashboardData").

```typescript
export function useJournalWidget() {
  const today = getLocalDateString();
  const { data, error, isLoading } = useSWR(
    `/api/journal/today?date=${today}`,
    fetcher,
    { keepPreviousData: true, revalidateOnFocus: true }
  );
  return {
    entry: data?.entry ?? null,
    streak: data?.streak ?? 0,
    onThisDay: data?.on_this_day ?? [],
    error,
    isLoading,
  };
}
```

Use `revalidateOnFocus: true` so widget refreshes when user navigates back from journal page (Pitfall 4 from research).
