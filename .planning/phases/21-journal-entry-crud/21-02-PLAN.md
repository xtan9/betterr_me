**Step 2: Create `components/journal/journal-entry-modal.tsx`.**
The main modal that wires all Plan 01 components together:
- `"use client"` directive
- Imports:
  - `Dialog`, `DialogContent`, `DialogHeader`, `DialogTitle` from `@/components/ui/dialog`
  - `JournalEditorLoader` from `./journal-editor-loader`
  - `JournalEditorSkeleton` from `./journal-editor-skeleton`
  - `MoodSelector` from `./journal-mood-selector`
  - `JournalSaveStatus` from `./journal-save-status`
  - `JournalDeleteDialog` from `./journal-delete-dialog`
  - `useJournalEntry` from `@/lib/hooks/use-journal-entry`
  - `useJournalAutosave` from `@/lib/hooks/use-journal-autosave`
  - `useTranslations` from `next-intl`
  - `useState`, `useCallback`, `useRef`, `useEffect` from `react`
  - `toast` from `sonner`
  - `getLocalDateString` from `@/lib/utils`

- Props interface:
  ```typescript
  interface JournalEntryModalProps {
    open: boolean;
    onOpenChange: (open: boolean) => void;
    date: string;  // YYYY-MM-DD format
  }
  ```

- Component logic:
  1. `const t = useTranslations();`
  2. `const { entry, isLoading, mutate } = useJournalEntry(date);` — fetches existing entry for the date
  3. `const { saveStatus, scheduleSave, flushNow, savedEntryId } = useJournalAutosave(entry?.id ?? null, date);`
  4. Track local mood state: `const [mood, setMood] = useState<number | null>(null);`
  5. Track dirty flag: `const [isDirty, setIsDirty] = useState(false);`
  6. Track latest content ref: `const contentRef = useRef<Record<string, unknown> | null>(null);`
  7. Track word count: `const [wordCount, setWordCount] = useState(0);`

  8. Sync mood from loaded entry: `useEffect(() => { if (entry) setMood(entry.mood); }, [entry]);`

  9. `handleEditorUpdate` callback:
     - Store content in contentRef
     - Set word count
     - If not dirty, check if content actually differs from loaded entry (skip initial onUpdate from Tiptap — Pitfall 5). Set isDirty to true only on real changes.
     - If isDirty, call `scheduleSave({ content: json, mood, word_count: wordCount })`

  10. `handleMoodChange` callback:
      - `setMood(newMood)` (convert 0 to null for API)
      - If isDirty or entry exists (meaning we have content to save), call `scheduleSave({ content: contentRef.current ?? entry?.content, mood: newMood || null, word_count: wordCount })`

  11. `handleDelete` callback:
      - Call `fetch(\`/api/journal/${entry!.id}\`, { method: "DELETE" })`
      - On success: `mutate()` to invalidate SWR cache, `toast.success(...)`, `onOpenChange(false)`
      - On error: `toast.error(...)`

  12. `handleOpenChange` callback:
      - If closing (open=false): call `flushNow()` to save any pending changes, then call `onOpenChange(false)`
      - If opening: call `onOpenChange(true)`

  13. Determine title: `entry ? t("journal.editEntry") : t("journal.newEntry")`
  14. Format date for display: create a date formatter that shows readable date (e.g., "February 22, 2026")

- Render:
  ```tsx
  <Dialog open={open} onOpenChange={handleOpenChange}>
    <DialogContent className="w-full sm:max-w-[70vw] max-h-[85vh] flex flex-col">
      <DialogHeader className="flex-shrink-0">
        <div className="flex items-center justify-between">
          <div>
            <DialogTitle>{title}</DialogTitle>
            <p className="text-sm text-muted-foreground">{formattedDate}</p>
          </div>
          <div className="flex items-center gap-2">
            <JournalSaveStatus status={saveStatus} />
            {entry && <JournalDeleteDialog onDelete={handleDelete} />}
          </div>
        </div>
      </DialogHeader>

      {/* Mood selector — above editor per Claude's discretion */}
      <div className="flex-shrink-0 py-2">
        <MoodSelector value={mood} onChange={handleMoodChange} />
      </div>

      {/* Editor area — grows to fill space */}
      <div className="flex-1 overflow-y-auto min-h-0">
        {isLoading ? (
          <JournalEditorSkeleton />
        ) : (
          <JournalEditorLoader
            content={entry?.content ?? null}
            onUpdate={handleEditorUpdate}
          />
        )}
      </div>

      {/* Word count footer */}
      <div className="flex-shrink-0 pt-2 border-t text-xs text-muted-foreground">
        {t("journal.wordCount", { count: wordCount })}
      </div>
    </DialogContent>
  </Dialog>
  ```

Note on the Dialog sizing: use `w-full sm:max-w-[70vw]` to be responsive (full width on mobile, 70% on desktop — Pitfall 6 fix).
