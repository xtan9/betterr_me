<feature>
  <name>Plaid Webhook JWT/ES256 Verification</name>
  <files>lib/plaid/webhooks.ts, tests/lib/plaid/webhooks.test.ts</files>
  <behavior>
    `verifyPlaidWebhook(body: string, plaidVerificationHeader: string): Promise<boolean>`

    Verification steps (all must pass, return false on any failure):
    1. Decode JWT protected header without verifying — extract `kid` and `alg`
    2. Reject if `alg` !== 'ES256'
    3. Fetch JWK from Plaid using `kid` via `plaidClient.webhookVerificationKeyGet({ key_id: kid })`
    4. Import JWK using `jose.importJWK()`
    5. Verify JWT signature using `jose.jwtVerify()` with `maxTokenAge: '5 min'`
    6. Compute SHA-256 hash of `body` string
    7. Compare hash to `payload.request_body_sha256` claim
    8. Return true only if all steps pass

    Test cases:
    - Valid JWT + correct body hash → true
    - Valid JWT + wrong body hash → false
    - Expired JWT (iat > 5 min ago) → false
    - Wrong algorithm (RS256) → false
    - Invalid signature → false
    - Missing Plaid-Verification header → false
    - Plaid key fetch failure → false

    Implementation notes:
    - Use `jose` library (already installed in Plan 01) for JWT operations
    - Use Node.js `crypto.createHash('sha256')` for body hashing
    - Mock `plaidClient.webhookVerificationKeyGet()` in tests using vi.mock
    - Generate test ES256 keys using jose `generateKeyPair('ES256')` in test setup
    - The function should accept a Plaid client instance as parameter (dependency injection for testability) rather than creating one internally
  </behavior>
  <implementation>
    Function signature: `async function verifyPlaidWebhook(body: string, plaidVerificationHeader: string, plaidClient: PlaidApi): Promise<boolean>`

    Wrap entire function in try-catch — any error during verification returns false (fail closed).

    Use `jose.decodeProtectedHeader()` to read kid/alg without verifying (needed to select the right key).
    Use `jose.importJWK()` to import the JWK from Plaid's key endpoint.
    Use `jose.jwtVerify()` with `maxTokenAge: '5 min'` to verify signature and check expiry.
    Use `crypto.createHash('sha256').update(body).digest('hex')` for body hash comparison.
  </implementation>
</feature>
