## 1. DESIGN PRINCIPLES

### 1.1 Core Philosophy

| Principle | Implementation |
|-----------|----------------|
| **Zero Friction** | 1-tap habit completion, no modals for quick actions |
| **Progress Visible** | Streaks prominent, completion % always shown |
| **Compassionate** | Encouraging copy, no shame for missed days |
| **Glanceable** | Dashboard scannable in <5 seconds |

### 1.2 Visual Hierarchy

1. **Primary:** Today's habits (toggle area)
2. **Secondary:** Progress stats (streaks, completion %)
3. **Tertiary:** Tasks, motivation message

### 1.3 Color System

```
Primary Actions:    emerald-500 (#10b981) - Completions, success
Streaks:            orange-500 (#f97316) - Fire, achievement
Missed:             slate-300 (#cbd5e1) - Gentle, not harsh red
Background:         white / slate-950 (dark mode)
Text Primary:       slate-900 / slate-50 (dark mode)
Text Secondary:     slate-500 / slate-400 (dark mode)
```

**Habit Categories:**
```
Health:       rose-500
Wellness:     purple-500
Learning:     blue-500
Productivity: amber-500
Other:        slate-500
```
